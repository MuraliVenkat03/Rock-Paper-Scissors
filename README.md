✊ Rock Paper Scissors – Web Game
📌 Project Description

This is a browser-based Rock Paper Scissors game built using HTML, CSS, and JavaScript. The game allows a user to play against the computer, where the computer generates a random choice and the winner is determined based on classic game rules.

🚀 Features

User vs Computer gameplay

Random computer choice generation

Real-time result display

Score tracking system

Interactive and responsive UI

Game reset functionality

🛠 Technologies Used

HTML – Page structure

CSS – Styling and layout

JavaScript – Game logic and randomization

🧠 Concepts Applied

DOM Manipulation

Event Listeners

Math.random() for computer choice

Conditional logic

Score management

🎯 Objective

The goal of this project was to practice front-end fundamentals and implement a simple interactive browser game using pure JavaScript without any frameworks.
